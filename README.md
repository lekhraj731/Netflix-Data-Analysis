# Netflix Data Analysis

## Project Overview

This project analyzes Netflix movies and TV shows using Python. The objective is to identify trends in content type, countries, genres, ratings, and content growth.

## Dataset

Netflix Titles Dataset from Kaggle.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed

### 1. Data Cleaning
Handled missing values and checked data quality.

### 2. Duplicate Check
Verified that no duplicate records exist.

### 3. Movies vs TV Shows Analysis
Compared the number of movies and TV shows available on Netflix.

### 4. Top Countries Analysis
Identified countries producing the most Netflix content.

### 5. Content Growth Analysis
Analyzed how Netflix content increased over the years.

### 6. Monthly Content Addition Analysis
Studied content addition trends across months.

### 7. Genre Analysis
Identified the most popular genres on Netflix.

## Key Insights

- Movies dominate Netflix's content library.
- The United States contributes the most content.
- Drama is one of the most common genres.
- Netflix experienced rapid growth after 2015.
- Content is added consistently throughout the year.
